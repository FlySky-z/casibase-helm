# Casibase Helm Chart

This Helm Chart is used to deploy the Casibase application.

## Directory Structure
- Chart.yaml: Helm chart metadata
- values.yaml: Configurable parameters
- templates/: Kubernetes resource templates

## Installation
```bash
helm install casibase .
```

## Upgrade
```bash
helm upgrade casibase .
```

## Uninstall
```bash
helm uninstall casibase
```
